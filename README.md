# Vim Tutorial

Tutorial and resources on the text editor Vim

## Compile PDF

```Shell
$ git clone git@github.com:erictleung/tutorial-vim.git
$ cd tutorial-vim
$ make
```

## Compiling Requirements

- `pandoc`
- `pdflatex`
